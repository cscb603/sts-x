✦ 核心功能

为 AI 设计的代码搜索

默认输出 JSON，AI Agent 可直接消费。三种搜索模式覆盖全部场景。

🧩

AST 感知切块

基于 tree-sitter 解析语法树，按函数、类、方法返回完整代码块，而非零散行匹配。

⚡

极速 BM25

Tantivy 倒排索引引擎，毫秒级响应。17MB 单文件，零外部依赖。

🔌

MCP 协议服务

内置 axum HTTP 服务器，POST /search 接口，任何支持 MCP 的 AI Agent 可直接调用。

📦

零依赖部署

无需 Python、Node.js、数据库。一个二进制扔到服务器就能用。

🔍

三种搜索模式

Code（代码语义）、Filename（文件名匹配）、All（全文件内容），覆盖全部搜索场景。

🧠

AI 原生输出

默认 JSON 格式，含分数、路径、签名、完整代码。也可用 -H 切换人类友好模式。

终端示例 — JSON 输出（AI 模式） 
# 索引项目
$ sts-x index .

# 搜索代码（默认 JSON，给 AI 消费）
$ sts-x search "token verification"
{
  "query": "token verification",
  "results": [{
    "score": 0.97,
    "path": "src/auth/jwt.rs",
    "signature": "fn verify_token(token: &str)",
    "code": "pub fn verify_token(...) { ... }"
  }],
  "total_hits": 5,
  "search_time_ms": 1
}

# 搜索文件名
$ sts-x search "config" -f

# 搜索所有文件内容
$ sts-x search "TODO" --all

# 启动 MCP 服务
$ sts-x serve --path .
# → AI Agent 可通过 POST http://127.0.0.1:9876/search 调用
✦ 快速上手

三步开始使用

不需要任何配置，下载即可运行。

1

下载

从 Releases 下载 sts-x-macos（17MB 单文件）

wget https://github.com/xtap/sts-x/releases/latest/download/sts-x-macos
2

安装

放入系统路径即可全局使用

sudo cp sts-x-macos /usr/local/bin/sts-x && sts-x --help
3

搜索

索引项目后即可搜索

sts-x index . && sts-x search "你的查询"
