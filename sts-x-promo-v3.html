它不是「降维打击整个赛道」

说句实话：STS-X 是词法 BM25 检索，不是语义/向量搜索。强在「已知符号、已知模式」的精准定位；想靠意思找「那个做鉴权的函数」，那是 Sourcegraph 的活。

但它把 AI 老一套「grep + 整文件 Read」真真切切降维了——省下的 80–92% 是实打实的 API 费用，可复现、可审计。
